# 🚢 Titanic Survival Prediction using Scikit-Learn

## 📌 Project Overview

This project develops a machine learning model to predict passenger survival on the Titanic dataset using Python and Scikit-Learn.

The notebook follows a structured data science workflow, beginning with data loading and exploration, followed by exploratory data analysis (EDA), data cleaning, feature engineering, model building, and evaluation.

The objective is to understand the relationship between passenger characteristics and survival while building a predictive classification model.

---

# 🎯 Objectives

- Explore the Titanic dataset.
- Perform Exploratory Data Analysis (EDA).
- Clean and preprocess the data.
- Engineer features suitable for machine learning.
- Train a Logistic Regression classifier.
- Evaluate the model using classification metrics.

# 📊 Exploratory Data Analysis (EDA)

The notebook includes exploratory analysis to better understand the dataset before model training.

The analysis includes:

- Dataset inspection
- Missing value analysis
- Survival distribution
- Feature visualization using graphs
- Relationship between important features and survival

EDA helps identify useful patterns and prepare the data for machine learning.

# 🧹 Data Cleaning & Feature Engineering

The notebook performs data preprocessing before model training, including:

- Handling missing values

- Selecting useful features
- Preparing data for machine learning
- Feature scaling using StandardScaler

# 🤖 Machine Learning Model

The project uses **Logistic Regression** from Scikit-Learn for binary classification.

The workflow includes:

- Train-Test Split
- StandardScaler
- Logistic Regression model training
- Prediction on test data
